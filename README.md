# Basic Calculator

## Overview

This project is a responsive web-based calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides an interactive user experience through a clean and modern user interface.

## Features

* Addition, Subtraction, Multiplication, and Division operations
* Real-time display updates
* Clear screen functionality
* Delete/Backspace functionality
* Keyboard support for faster input
* Responsive design for different screen sizes
* Smooth hover effects and user-friendly interface
* Error handling for invalid expressions

## Technologies Used

* HTML5
* CSS3
* JavaScript
